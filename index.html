Sobre el proyecto

Esta aplicación permitirá conectar a diferentes personas que presten diferentes tipos de servicios, como lo son, tutorías académicas, cuidado de personas y/o de mascotas, entre otras, con personas que necesiten estos servicios. Los prestadores del servicio recibirán, además del pago acordado, un puntaje, dado por el comprador, que permitirá que este prestador aumente su reputación o calidad del servicio en la plataforma. Para la publicación del servicio se necesitará un registro donde se pidan los datos básicos de la persona, así como el documento de identidad, esto para generar una mayor seguridad para el usuario de que la persona que presta si es quien dice ser. El sistema permitirá una búsqueda del servicio solicitado, y además unos filtros predefinidos, tales como puntaje del prestador del servicio, localización, tipos de servicio, precio, etc.
